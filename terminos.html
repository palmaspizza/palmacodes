Términos y Condiciones

Última actualización: 17 de abril de 2026

Importante: Al contactarnos a través de WhatsApp o cualquier medio en este sitio, aceptas estos términos. Si no estás de acuerdo, por favor no utilices nuestros servicios.

1. Definiciones

  • "Palma Codes", "nosotros" o "el desarrollador": Diego Palma, creador del sistema de reservas.
  • "Sistema": La aplicación web de reservas para barberías desarrollada por Palma Codes.
  • "Cliente" o "tú": La persona que adquiere o utiliza el sistema.
  • "Usuario final": Los clientes de tu barbería que utilizan el sistema para hacer reservas.

2. Sobre el servicio

Palma Codes ofrece un sistema de reservas web personalizado para barberías. El servicio incluye:

  • Desarrollo y configuración inicial del sistema
  • Personalización con nombre, logo y servicios de tu barbería
  • Panel de administración para gestionar reservas
  • Soporte técnico básico vía WhatsApp

3. Modelo de pago

Pago único: El sistema tiene un costo de $30.000 CLP (pesos chilenos) como pago único. No hay mensualidades ni costos recurrentes ocultos.

El pago se realiza:

  • 50% al momento de confirmar el pedido (para iniciar el desarrollo)
  • 50% al entregar el sistema funcionando

4. Alcance y limitaciones

4.1 Lo que incluye:

  • Sistema funcional con las características descritas en el sitio
  • Personalización de marca (colores, nombre, servicios)
  • Configuración inicial de horarios y servicios
  • Soporte por WhatsApp durante las primeras 2 semanas
  • Corrección de errores técnicos del sistema

4.2 Lo que NO incluye:

  • Hosting o dominio personalizado (el sistema se aloja en Firebase gratuito)
  • Modificaciones estructurales mayores después de la entrega
  • Capacitación presencial (toda la documentación es digital)
  • Integraciones con sistemas de terceros no especificados
  • Garantía sobre la disponibilidad de Firebase (servicio de terceros)

5. Propiedad intelectual

El código base del sistema es propiedad de Palma Codes. Al adquirir el sistema:

  • Obtienes una licencia de uso perpetuo para tu barbería
  • No puedes revender, sublicenciar o distribuir el sistema
  • No puedes utilizar el código para crear un producto competitivo
  • Tu marca, logo y datos de clientes son de tu propiedad

6. Responsabilidades del cliente

Al utilizar el sistema te comprometes a:

  • Proporcionar información veraz sobre tu negocio
  • No utilizar el sistema para actividades ilegales
  • Mantener la confidencialidad de tus credenciales de acceso
  • Respetar la privacidad de los datos de tus clientes finales
  • No intentar acceder al código fuente o modificar el sistema

7. Limitación de responsabilidad

Palma Codes no se hace responsable por:

  • Pérdida de datos por fallos en Firebase (servicio de Google)
  • Daños indirectos, pérdida de beneficios o interrupción del negocio
  • Problemas causados por modificaciones no autorizadas
  • Uso indebido del sistema por parte del cliente o usuarios finales
  • Fallos en la conectividad a internet del cliente o sus usuarios

La responsabilidad máxima de Palma Codes está limitada al monto total pagado por el sistema.

8. Soporte técnico

El soporte incluye:

  • Respuesta a consultas por WhatsApp en horario hábil (L-V, 10:00-19:00)
  • Corrección de errores del sistema
  • Ayuda con configuraciones básicas

El soporte no incluye desarrollo de nuevas funcionalidades, que se cotizarán por separado si se requieren.

9. Cancelación y reembolsos

  • Si cancelas antes de iniciar el desarrollo: reembolso del 100%
  • Si cancelas durante el desarrollo: reembolso del 50% (se retiene el anticipo)
  • Una vez entregado el sistema: no hay reembolsos por ser un producto digital personalizado

10. Modificaciones a los términos

Podemos actualizar estos términos ocasionalmente. Los cambios entrarán en vigor al publicarse en esta página. Te recomendamos revisar esta página periódicamente.
